# GrooveLinx – Where Bands Lock In

GrooveLinx is a web app for modern jam bands (Dead / Phish / Goose / WSP / ABB style) who want to tighten up without killing the vibe.

- Shared **Song Library & Song DNA** (key, BPM, structure, lead, harmonies)
- Band-voted **North Star** reference recordings
- **Best Shot vs North Star** comparisons for your own takes
- **Woodshed** tools: stems, practice tracks, harmony parts, rehearsal notes
- Intelligent **setlists**, **calendar**, and **gig logistics** in one place
- Built-in **tuner** and **metronome** for quick checks

This repo powers the public marketing site at **GrooveLinx.com**.  
The live demo app currently runs at: `https://drewmerrill.github.io/deadcetera/`.
